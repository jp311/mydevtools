在文件模板中，可以使用文本、代码、注释和预定义变量。预定义变量列表如下。在模板中使用这些变量时，它们稍后会在编辑器中扩展为相应的值br>
也可以指定自定义变量。自定义变量使用以下格式： ${VARIABLE_NAME}, 其中 VARIABLE_NAME 是变量名 (例如, ${MY_CUSTOM_FUNCTION_NAME}). 在 IDE 创建包含自定义变量的新文件之前，您会看到一个对话框，您可以在其中定义模板中自定义变量的值

通过使用 #parse 指令, 可以使用 Includes指令包含选项卡中的其他模板. T若要包含模板，请在引号中将模板的全名指定为参数 (例如： #parse("File Header").
预定义变量列表
${PACKAGE_NAME}   当前包名
${IMPORTS}   业务类所有import 字符串
${ORIGINAL_NAME}   业务类名
${CLASS_FIELDS}   业务类所有字段
${DATE}   当前系统日期
${DAY}   当前日期的天
${DAY_NAME_SHORT}   当月今天的日期
${DAY_NAME_FULL}   当前日期名称的前三个字母（星期一、星期二等）
${DIR_PATH}   新文件目录的路径（相对于项目根目录）
${DS}   美元符号 （$）。此变量用于转义美元字符，因此不会将其视为模板变量的前缀。
${FILE_NAME}   文件名称
${HOUR}   当前小时
${MINUTE}   当前分钟
${SECOND}   当前秒钟
${MONTH}   当前月份
${MONTH_NAME_SHORT}   当前月前3个字母 (Jan, Feb, and so on)
${MONTH_NAME_FULL}   当前月全称 (January, February, and so on)
${NAME}   文件名称
${ORGANIZATION_NAME}   在项目设置的组织名称
${PRODUCT_NAME}   IDE产品名称
${PROJECT_NAME}   项目名称
${TIME}   当前系统时间
${USER}   当前用户的系统登录名
${YEAR}   当前年
